1 前言

2 CPU版安装

2.1 本机环境
  • Anaconda
  • Windows 10 x64bit
  • 笔记本渣渣GPU 1.0算力，CPU版不需要
  • 已装有vs2017

3 GPU版安装

3.1 本机环境
  • Anaconda
  • Windows 10 x64bit
  • 台式 GPU：GTX 1060 6G 内存：16G
  • 已装有vs2017
  • CUDA 10.0
  • CUDNN 7 x64bit
3.2 安装GPU版

　　先去英伟达官网下载cuda并安装。
　　接着下载cudnn放到cuda安装目录下的cuda文件夹下，并在系统环境变量中添加cudnn的bin路径。
　　然后创建环境python环境，进入环境后，pip安装tensorflow2.0.0 gpu版即可。

（1）在创建模型时，出现缺少库的问题：Failed to import pydot。
　　安装pydot和graphviz。结果运行后还是有问题。尝试安装pip install pydot-ng。
　　接着问题变成：InvocationException: GraphViz’s executables not found。
　　先尝试安装pip install pydotplus。安装后还是GraphViz的问题。
　　下载安装GraphViz2.38，然后在开头添加环境变量代码（其实可以一劳永逸设置在系统环境变量里里）

（2）在执行模型时发生错误：ValueError: ‘mnist model’ is not a valid scope name.
　　原因：model的名字被命名成了“mnist model”，中间有个空格，这是无效的命名，改为“mnist_model”就正常了。顺便将batch改小，由64改为8。运行两个epoch方便测试。
